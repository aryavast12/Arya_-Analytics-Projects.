# Hospital Cost Driver Analysis

## 🎯 Objective
To analyze the relationship between patient wait times, treatment costs, and overall profitability across various hospital departments (ER, ICU, and OPD). 

## 🛠️ Tools Used
* **Microsoft Excel:** Pivot Tables, Slicers, Bar Charts, and Scenario Modeling.

## 📊 Project Overview
This project uses a hospital dataset to track metrics by department and doctor. I created an interactive dashboard to visualize these relationships and conducted a simple "What-If" analysis to project the impact of rising costs.

### Key Features & Insights:
* **Departmental Breakdown:** Analyzed Average Wait Time vs. Average Treatment Cost using Pivot Tables. The ICU shows the highest wait times (avg 91 mins) and highest costs, while the OPD is the most efficient.
* **Interactive Dashboard:** Implemented slicers to allow users to filter data dynamically by specific Departments or Doctors.
* **Scenario Modeling:** Calculated the financial impact of a 5% increase in total costs, which demonstrated a projected profit decrease of ₹3,560, highlighting the sensitivity of hospital margins.
